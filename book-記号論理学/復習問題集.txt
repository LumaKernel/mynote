+ スコーレム標準形とはなにか
  - 論理式に対する変形で，∃(∀)が全部前に来て，∀(∃)を含まない論理式．
  変形は意味論的に同値とは限らないが，恒真性(恒偽性)は同値である変形．
+ エルブラン (Herbrand) 解釈 とはなにか
  - 項を項のまま記号として解釈するような解釈．基本述語に対する解釈が決まれば，全体の解釈も定まる
+ エルブラン解釈の嬉しさはなにか
  - 項の内容に踏み込まず，基本述語の真偽だけを見ることができる．つまり，命題論理に近い感じになる．
充足可能ならエルブラン解釈が存在するため，エルブラン解釈だけを考慮すれば良くなる．
+ エルブランの定理とはなにか
  -
  エルブラン標準形に変形したとき，その恒真性(恒偽性)を保存するような，いくつかの代入が存在するというもの．
+ 決定可能な一階述語論理の論理式とはどのようなものか(本の中で解説されているもの)
  - ∀∃標準形，∀が来て∃が来て，もう量化子は来なくて，全体として自由変項がなく，演算子を含まないもの
+ 単項一階述語論理とはなにか
  - 述語は単項述語のみで，演算子を含まない一階述語論理．すべての論理式が決定可能 
+ タブロー，また，それによる証明とはなにか
  - 木の形をしてる形式的な証明，背理法に基づく
+ 証明論と意味論の違いはなにか
  - 証明論は論理式が何を意味するのか，に踏み込まず，形式的な変形のみを許す
  意味論は論理式を真理値0,1や真理関数に読み替える
+ ヒルベルト流証明体系 (Hilbert proof system) とはなにか
  - 推論規則を最小限 [(MP)と(GEN)] にした証明体系で，公理はそれなりに多い．
+ 自然演繹とはなにか
  - ヒルベルト流では単純な論理式(図式)を証明するのも難しく，それ自体が何らかの意味を与えてしまっているのではないかという
  懸念から，自然に演繹できる証明体系．dischargeといった操作がある．公理は最小限 [(ID)] で推論規則を主に使う．
+ シーケント計算 (sequent calculus) とはなにか
  - 証明体系の証明自体を対象とし図式する．それ自体は自然演繹に見た目が似ている．
+ シーケント計算の嬉しさはなにか
  - ヒルベルト流と自然演繹，といった異なる証明体系を同じ次元で扱えること．
  - 推論規則などが従来より完結に書けること [(∀I)など]
+ ゲンツェン (Gentzen) 流シーケント計算とはなにか
  - シーケント計算の証明体系のひとつで，基本的には推論規則からなる．推論規則にはそれなりに対称性がある．
+ カット除去定理とはなにか
  - LKの規則 (CUT) は LK-CUT の許容規則 (admissible rule) である，ということを言っている定理
+ カット除去定理の嬉しさ，もしくはカット規則の利点/欠点とはなにか
  - カット除去定理により，無矛盾性などが簡単に証明できる
  - カット規則は自由度が高く，帰納法などで証明するときに扱いづらい
  - カット規則は自由度が高いので，LKで証明するときには使いやすい
+ グリベンコの定理とはなにか
  - Γ ⊢LK Δ ⇔ ¬Δ*, Γ* ⊢LJ ⊥
  - ただし φ* は黒田否定変換，全称量子化の直後に二重否定をいれていく
+ ゲーデルの完全性定理 (Gödel's completeness theorem) とはなにか
  - ⊢TAB φ ⇔ ⊨φ
+ ヘンキンの定理 (Henkin's theorem) とはなにか
  - Γ⊢/TAB ⇒ Γは可算領域で充足可能
  - TABの完全性の証明で使える
+ コンパクト定理とはなにか
  - 任意のΓ'⊂Γで|Γ'|が有限であるものについて，Γ'は充足可能 ⇒ Γは充足可能
+ 無限列を含むかもしれないシーケントが，(意味論的に/証明論的に) 妥当/証明可能であるとはどう定義されるか
  - 意味論では，無限列内のすべての論理式について，真だと解釈するような，一つの解釈が存在すること
  - 証明論では，無限列の有限部分列で，証明可能なものが一つでも存在すること
+ 充満した列とはなにか
  - 列のうち，その列に含まれる論理式について，その論理式からタブローによって得られる可能性のある論理式が，
  同じ列に含まれているというもの
  - その部分列から始まる任意のタブローについて，登場する論理式は元の列に含まれている，というもの
+ レーベンハイム・スコーレムの定理 (Löwenheim–Skolem theorem: LST) とはなにか
  - 充足可能なら，可算領域で充足可能である，ということを言っている
  - ヘンキンの定理とTABの健全性からただちに証明できる
